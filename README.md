# Josef-Umzugsservice Website

Diese Dateien sind bereit für GitHub Pages.

## Deployment
1. Repository erstellen auf GitHub
2. Alle Dateien aus diesem ZIP in das Repo hochladen (nicht die ZIP)
3. Settings → Pages → Branch: main, Folder: / (root)
4. Custom domain: josef-umzugsservice.de (falls gewünscht)
